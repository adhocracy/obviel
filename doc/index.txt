Obviel: Object/View/Element for JavaScript
==========================================

Obviel is a model/view abstraction for JavaScripts that helps
you improve the structure of your jQuery-based JavaScript applications.

Obviel features:

* JSON-centric.

* versatile but unobtrustive.

* supports true REST development patterns.

* loose coupling between client code and server code.

* client-side templating integration.

* powerful client-side form library.

Curious? Want to know more? Go on to the :doc:`introduction <intro>`
or view the :doc:`table of contents <toc>`.
